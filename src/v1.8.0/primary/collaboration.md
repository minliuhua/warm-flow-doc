# 或、票、会签

::: tip 
- 审批过程中可能存在，一人或者多人审批的情况，不同情况对应不同协作类型（会签、票签和或签）

:::

## 1、流程协作规则
> 或签：一名办理人审批通过，审批节点才会通过  
> 票签：满足设定的通过率，部分审批人审批通过，审批节点才会通过  
> 会签：所有审批人审批通过，审批节点才会通过
