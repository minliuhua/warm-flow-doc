1. 支持常规的流程流转，比如跳转、回退、审批和任意跳转
2. 支持转办、终止，任务最终回到发起人
3. 支持或签（会签和票签开发中）
4. 业务项目可不依赖流程设计器，组件会生成流程图片
5. 支持角色、部门和用户等权限配置
6. 支持监听器，参数传递，动态权限 
7. 支持多租户
8. 支持互斥网关，并行网关
9. 支持条件表达式，可扩展
10. 支持不同orm框架系统使用，支持不同orm框架和数据库扩展
11. 同时支持spring和solon
12. 兼容java8和java17,理论11也可以
13. 官方提供基于ruoyi-vue封装实战项目，很实用
